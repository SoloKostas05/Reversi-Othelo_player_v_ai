

Περιγραφή

Όλα τα απαραίτητα για την εκτέλεση του προγράμματος περιέχονται στο
αρχείο Reversi.java. Ο κώδικας είναι εξ ολοκλήρου γραμμένος σε Java και
περιλαμβάνει το παιχνίδι Reversi/Othello, καθώς και τον αλγόριθμο
MiniMax που χρησιμοποιείται για τον αντίπαλο υπολογιστή (AI).

Οδηγίες Εκτέλεσης

1.  Ανοίξτε το Command Prompt (cmd) μέσα στον φάκελο όπου βρίσκεται το
    αρχείο Reversi.java.

2.  Πληκτρολογήστε τις παρακάτω εντολές:

    javac Reversi.java java Reversi

3.  Το παιχνίδι θα ξεκινήσει αμέσως μετά τη μεταγλώττιση.

Επιλογές Παιχνιδιού (pregame input)

Κατά την εκκίνηση μπορείτε να επιλέξετε:

1 → Παίζετε ως Μαύρος (Black, πρώτος παίκτης – MAX) 2 → Παίζετε ως
Λευκός (White, δεύτερος παίκτης – MIN) 0 → Κρυφή επιλογή / test mode: Ο
αλγόριθμος παίζει ενάντια στον εαυτό του (AI vs AI)

Κανόνες & Κινήσεις

-   Οι κανόνες του παιχνιδιού είναι οι επίσημοι και γνωστοί κανόνες του
    Reversi/Othello.

-   Για να εκτελέσετε μια κίνηση, πληκτρολογήστε τη γραμμή και τη στήλη
    όπου θέλετε να τοποθετήσετε το πιόνι σας, χωρισμένα με κενό, όπως
    στο παρακάτω παράδειγμα:

    4 5

-   Στη συνέχεια πατήστε Enter.

-   Εάν η κίνηση δεν είναι έγκυρη, θα ενημερωθείτε και θα σας ζητηθεί να
    επιλέξετε μια νέα.

-   Το παιχνίδι ολοκληρώνεται όταν κανένας παίκτης δεν έχει διαθέσιμες
    έγκυρες κινήσεις.

Νικητής

Νικητής είναι ο παίκτης που, στο τέλος του παιχνιδιού, έχει τα
περισσότερα πιόνια στο ταμπλό.
